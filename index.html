What is "The Duxer"?

"The Duxer" is an audio player build with the new__ Media Elements API included in HTML5. For now this is a basic version. You can see the tutorial of how this player was built in my Youtube Channel.

Action!

You can see "The Duxer" in action on: http://rfdz.mx/tutoriales/demos/theduxer

Support

The Duxer works in the latest versions of web browsers. The player also was tested in iPod/iPad/iPhone.
